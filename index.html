Истории пользователей

  1. Мне, как пользователю, нужно выбирать, за что играть (X/O), для начала хода первым или вторым.
  2. Мне, как пользователю, нужно менять сложность при игре против бота, для выбора подходящего уровня игры.
  3. Мне, как пользователю, нужно играть не только с ботом, но и с другом, для совместной игры.

Истории администратора сайта

  1. Мне, как администратору сайта, нужно указывать размеры поля игры для настройки игрового процесса.
  2. Мне, как администратору сайта, нужно добавлять несколько игровых полей для их независимой работы.
  3. Мне, как администратору сайта, нужно сохранять историю ходов текущей игры для анализа.

Истории разработчика приложения

  1. Мне, как разработчику приложения, нужен класс TicTacToe для управления поведением приложения.
  2. Мне, как разработчику приложения, нужен умный бот для игры против пользователя.
  3. Мне, как разработчику приложения, нужно легко добавлять новые функции в игру для её улучшения.
